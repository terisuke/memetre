# 👁️ めめトレ！楽しく視力UP

老眼・近眼の方に向けた、ブラウザで遊べる目のトレーニングアプリです。

> ⚠️ これは楽しく目を動かすための補助ツールです。老眼・近眼の根本治療ではありません。症状が気になる場合は眼科医に相談してください。

## 🎮 3つのミニゲーム

| ゲーム               | 内容                                                                | 所要時間 |
| -------------------- | ------------------------------------------------------------------- | -------- |
| 🏃‍♂️ **目で追え！**    | 動く的を目と指で追いかける眼球運動トレーニング。だんだん速くなる！  | 30秒     |
| 🔍 **ピント合わせ**  | 遠くと近くを交互に見る調節力トレーニング。反応速度を測定！          | 30秒     |
| 🅲 **ランンドルト環** | Cの切れ目の向きを当てる視力検査スタイルのトレーニング。10レベル制！ | 約1分    |

## 🕒 20-20-20ルールタイマー

20分ごとに20フィート（6m）先を20秒見る、目の休息ルールをサポートするビジュアルタイマー付き。

## 🎉 特徴

- **サウンドエフェクト** — 正解・不正解・クリア時の音（Web Audio API合成）
- **スコア履歴** — 直近7日間のスコアをバーチャートで表示
- **連続日数** — 毎日続けている日数をカウント（🔥連続N日目）
- **スマホ対応** — タッチ操作対応、レスポンシブCanvas
- **インストール不要** — ブラウザだけで完動、CDN依存のみ
- **データはローカル** — スコアは localStorage に保存、外部送信なし

## 🚀 使い方

以下のURLにアクセスするだけ：

**https://terisuke.github.io/memetre/**

または、このリポジトリをクローンして `index.html` をブラウザで開いてください。

```bash
git clone https://github.com/terisuke/memetre.git
open memetre/index.html
```

## 🛠️ 技術スタック

- HTML / CSS / JavaScript（単一ファイル）
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- Canvas API（ゲーム1・3）
- Web Audio API（サウンド合成）
- localStorage（スコア永続化）
